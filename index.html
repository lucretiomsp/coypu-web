coypu.tone

cascade syntax · polymetric · transpiler demo
tap play to start audio
rhythm statements create a sequencer bound to a track:
16 downbeats to: #bd · #rumba asRhythm to: #snare · '12caf1213' hexbeat to: #hat
parameter cascades attach to an existing track:
#snare notes: #(38 38 40); durations: #(1 1 2); level: 0.6
defaults — note 60, duration 1 step, level 0.5. arrays cycle per hit, independently.
