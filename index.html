coypu.tone

cascade syntax · polymetric · transpiler demo
tap play to start audio
rhythm statements create a sequencer bound to a track (folder):
16 downbeats to: #kick · #rumba asRhythm to: #snare · '12caf1213' hexbeat to: #ch
parameter cascades attach to an existing track:
#ch index: 2; notes: #(60 72 48); level: 0.4
index: picks the sample in the folder (positional). notes: sets playback pitch — 60 = original, 72 = octave up (2×), 48 = octave down. defaults — index 0, note 60, level 0.5. arrays cycle per hit.
